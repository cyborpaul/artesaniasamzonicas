
## Intalación del proyecto Artesanías Amazónicas

Para poder instalar este proyecto, es necesario contar con los siguientes requisitos:

- Intalar Composer la última versión a nivel global.
- VSC y algunos complementos(opcional).

## Intalación

- Clonar a la rama master o descargar de forma manual.
- Ejecutar "composer install" dentro de la consola de VSC para obtener la carpeta Vendor.
- Luego el archivo ".env.example" cambiar a ".env"
- Ejecutar "php artisan key:generate" dentro de la consola de VSC para generar la llave global.
- Crear la base de datos "artesaniasamzonicas".
- Ejecutar "php artisan migrate" dentro de la consola de VSC para tener las tablas para la base de datos.
